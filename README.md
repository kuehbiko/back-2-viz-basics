# back-2-viz-basics
Back 2 Viz Basics challenge submissions using Power BI dashboards and simple data engineering pipelines.

This challenge is originally designed for Tableau visualizations. Here are some changes I've made for personal practice:

1. Instead of downloading the .csv files provided, we will be scraping the data from the data source webpage
2. Instead of uploading the data directly into Power BI, it will be cleaned and explored in a jupyter notebook, then uploaded to a local postgres database
3. Instead of using Tableau, we will connect to the data and visualize it using Power BI desktop
